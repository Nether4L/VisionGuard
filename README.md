# VisionGuard

Легковесное фоновое C# приложение для Windows, помогающее снизить нагрузку на зрение при длительной работе за компьютером. Программа автоматически реализует популярное офтальмологическое правило **«20-20-20»**: каждые 20 минут напоминает перевести взгляд на объект в 20 метрах перед собой в течение 20 секунд.

## 🚀 Особенности

* **Полная автономность:** Приложение работает исключительно в фоновом режиме и не имеет главного окна, что предотвращает случайное закрытие и эконосит экранное пространство.
* **Минималистичный Tray-интерфейс:** Управление программой происходит через лаконичную иконку в системном трее Windows (возле часов).
* **Нативные уведомления:** Оповещения используют встроенный механизм операционной системы (`Balloon Tip`), аккуратно всплывая в углу экрана.
* **Удобное управление:** Быстрый выход из приложения доступен по правому клику мыши через контекстное меню.

## 🛠 Стек технологий

* **Язык программирования:** C# 15
* **Платформа:** .NET 10.0
* **Графический фреймворк:** Windows Forms (`ApplicationContext`, `NotifyIcon`, `ContextMenuStrip`)

## 📂 Архитектура и структура кода

Проект построен на базе кастомного контекста приложения, что позволяет избежать создания дефолтных графических форм:

* **`Program.cs`** — точка входа в приложение. Настраивает конфигурацию и запускает фоновый цикл обработки сообщений Windows через экземпляр `AppConfigContext`.
* **`AppConfigContext.cs`** — ядро приложения. Наследуется от `ApplicationContext`. Внутри инкапсулирована логика инициализации иконки трея, сборка контекстного меню, конфигурация фонового таймера и вызов системных уведомлений.

## ⚙️ Требования и запуск

1. Склонируйте репозиторий на локальный компьютер.
2. Откройте проект в Visual Studio 2022, JetBrains Rider или VS Code.
3. Соберите проект в конфигурации `Release` или `Debug`.
4. Запустите скомпилированный `.exe` файл. Приложение сразу появится в системном трее.

> ⚠️ **Важно:** Если после запуска таймера уведомления не отображаются на экране, проверьте параметры вашей операционной системы (*Параметры -> Система -> Уведомления*). Убедитесь, что для приложений разрешен показ уведомлений, а режим **«Не беспокоить» (Focus Assist)** отключен.
